# Brazilian CPF

Brazilian CPF é uma versão python contendo funcionalidades para validar e gerar CPF's brasileiros.

Como projeto de estudo de uso básico do Python, tem finalidade educadional do ponto de visto do uso de cálculos e algoritmos.

 Este projeto serve como exercício de apoio em aulas.

## Conceitos e técnicas utilizadas

O que o aluno encontra neste projeto:

- controle de fluxo;
    - desvio condicional;
    - laço de repetição;
- operadores matemáticos;
- manipulação de cadeia de caracteres;
- uso de bibliotecas (random);
- uso de orientação a objetos;
- I/O com escrita de arquivo em dispositivo de armazenamento;


## ⚖️ Aviso Legal e Conformidade

Este projeto tem fins estritamente **educacionais e de estudo lógico-matemático**. O uso de CPFs de terceiros ou a geração de documentos falsos sem autorização é crime previsto no **Código Penal Brasileiro**.

Abaixo, as principais referências legais sobre o uso indevido de documentos e dados de terceiros:

| Infração | Descrição Breve | Referência Legal |
| :--- | :--- | :--- |
| **Falsidade Ideológica** | Omitir ou inserir declaração falsa em documento público ou particular. | [Art. 299 do Código Penal](https://www.planalto.gov.br/ccivil_03/decreto-lei/del2848compilado.htm#art299) |
| **Falsa Identidade** | Atribuir-se ou atribuir a terceiro falsa identidade para obter vantagem. | [Art. 307 do Código Penal](https://www.planalto.gov.br/ccivil_03/decreto-lei/del2848compilado.htm#art307) |
| **Falsificação de Doc. Público** | Falsificar, no todo ou em parte, documento público ou alterar documento verdadeiro. | [Art. 297 do Código Penal](https://www.planalto.gov.br/ccivil_03/decreto-lei/del2848compilado.htm#art297) |
| **Estelionato** | Obter vantagem ilícita, em prejuízo alheio, induzindo ou mantendo alguém em erro. | [Art. 171 do Código Penal](https://www.planalto.gov.br/ccivil_03/decreto-lei/del2848compilado.htm#art171) |

> **Nota:** O tratamento de dados pessoais no Brasil também é regulamentado pela **LGPD (Lei nº 13.709/2018)**. Você pode consultar o texto integral da lei [neste link do Gov.br](https://www.planalto.gov.br/ccivil_03/_ato2015-2018/2018/lei/l13709.htm).
